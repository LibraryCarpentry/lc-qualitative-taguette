::: callout

This lesson can be combined with [QualCoder](https://librarycarpentry.github.io/lc-qualitative-qualcoder/) for a full-day workshop. We recommend the following structure:
1. Teach Taguette lesson as written through the [Deductive Coding](https://librarycarpentry.github.io/lc-qualitative-taguette/03-best-practices-for-qualitative-coding.html#deductive-coding) section.
2. Demonstrate how to [export Taguette project](https://librarycarpentry.github.io/lc-qualitative-taguette/05-saving-and-sharing-qualitative-data.html).
3. Begin at [Starting QualCoder](https://librarycarpentry.github.io/lc-qualitative-qualcoder/01-introduction-to-qualitative-data.html#starting-qualcoder) and import the Taguette project to QualCoder (`Project - Import - Taguette Import`) instead of creating a new project.
3. Continue the QualCoder lesson, abbreviating or skipping sections in [Best Practices for Qualitative Coding](./episodes/03-best-practices-for-qualitative-coding.html) that replicate the Taguette lesson and don't reference the software.

:::
